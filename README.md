1. Install dependencies
   ```sh
   $ npm install
   ```

2. Serve the presentation and monitor source files for changes
   ```sh
   $ npm start
   ```

3. Open <http://localhost:8000> to view your presentation

   You can change the port by using `npm start -- --port=8001`.
